# About

This is MyPlanner built for SD Hack.


## Technology
```
Frontend: vue2 + vuex + vue-router + webpack + ES6/7 + fetch + sass + flex + svg  

Backend: spring boot + MongoDB + spring mvc

Natural language processing: python
```

## Front end

```
cd frontend

npm install or yarn

npm run dev

```

## Back end
```
mongod  
run it with IntelliJ
```
